# ColorForKids
ColorForKids is a user-friendly web application built on the Next.js framework, designed to provide a searchable and genre-seperated database of coloring pages from popular children's movies. The digital assets are organized into seven unique categories for easy discovery and access. The application uses TailwindCSS for responsive styling and JavaScript for dynamic rendering, and is pre-configured with ESLint and Prettier for automated code formatting. ColorForKids was built using UC Riverside's Association for Computer Machinery's Next.js Hackpack for a production-ready environment.

# Prediction-ML-Project
This project explores passenger survival on the Titanic using machine learning models, and emphasizes statistical interpretability and reasoning. Through Python and scikit-learn, I implemented a logistic regression model and a decision tree to analyze the influence of linear and nonlinear modeling in predictions. To understand the model behavior, I created a grid of sample passengers to identify cases where the models disagreed. The project showcases bias-variance tradeoffs, feature interactions, and threshold effects dependent on the model used.

# Task_Scheduler
Through C++, Git, and GoogleTest, I collaborated in a team of four to implement a task manager, where users can input task data (titles, priority levels, due date, etc.) and the tasks will be displayed in the terminal, sorted based on priority, and includes a progress bar of a task's completion. I applied Object-Orientated design patterns to ensure code maintainability, and subsequently developed a comprehensive suite of unit tests using the GoogleTest framework to ensure 100% logic coverage and system stability. My team and I managed the full software development life cycle using GitHub for version control and easy collaboration.
